# 15 — Game Source Code

> Source code đã được reskin sẵn. Để chỉnh UI và build:

```bash
npm install
npm run dev      # Chạy dev server để chỉnh UI
npm run build    # Build ra dist/ khi hoàn tất
```

## Cấu trúc thư mục
- `src/scenes/Scene1.ts` — UI Màn 1
- `src/scenes/Scene2.ts` — UI Màn 2
- `public/assets/reskin/` — Assets đã inject sẵn
- `public/assets/data/game_manifest.json` — Manifest đã được cập nhật
